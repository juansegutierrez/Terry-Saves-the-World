Terry Saves the World

"Disarming intergalactic threats, one spreadsheet at a time."

A comedic, narrative-driven puzzle game built with pure HTML5 Canvas and Vanilla JavaScript.

"Terry Saves the World" is a comedic, narrative-driven puzzle game built from scratch using HTML5 Canvas and Vanilla JavaScript. It reimagines classic Minesweeper logic through the eyes of Terry, a mundane office worker tasked with disarming alien "Quantum Destabilizers" (while being denied hazard pay by HR).
